## Геймплей
- Управление подводной лодкой с помощью кнопок (вверх, вниз, влево, вправо)
- Атака торпедами
- Уклонение от скал, акул и кораблей
- Три уровня сложности
- Таблица рекордов

## Технологии
- Swift
- UIKit
- Storyboard
