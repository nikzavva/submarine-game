# The Submarine

Игра, где нужно управлять подводной лодкой, уклоняться от препятствий и уничтожать врагов.

## Геймплей
- Управление подводной лодкой с помощью кнопок (вверх, вниз, влево, вправо)
- Атака торпедами
- Уклонение от скал, акул и кораблей
- Три уровня сложности
- Таблица рекордов

## Технологии
- Swift
- UIKit
- Storyboard
